# 🌐 Web II — DSM · Fatec

Lista de exercícios de Flask da disciplina **Desenvolvimento Web II**, do curso de **Desenvolvimento de Software Multiplataforma (DSM)**.

O foco da lista foi o desenvolvimento de APIs e aplicações back-end com Python e Flask, abordando conceitos fundamentais de HTTP, autenticação, manipulação de dados e boas práticas de desenvolvimento web.

---

##  Conteúdo abordado

- Criação de rotas com **Flask**
- Métodos HTTP: `GET` e `POST`
- Parâmetros de URL, query strings e corpo da requisição
- **Cabeçalhos HTTP** (Request Headers) e autenticação simples com `X-Api-Key`
- Retorno de respostas com `jsonify` e códigos de status HTTP
- Estruturação de projetos Flask

---

##  Tecnologias

- [Python 3.x](https://www.python.org/)
- [Flask](https://flask.palletsprojects.com/)

---

##  Como executar

### 1. Clone o repositório

```bash
git clone https://github.com/geconstancio/WEBII_DSM_Fatec.git
cd WEBII_DSM_Fatec
```

### 2. Crie e ative um ambiente virtual

```bash
python -m venv venv

# Linux / macOS
source venv/bin/activate

# Windows
venv\Scripts\activate
```

### 3. Instale as dependências

```bash
pip install flask
```

### 4. Execute a aplicação

```bash
python app.py
```
